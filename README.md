# Cloud Crafter
An interactive recipie maker for DIY vape juice via CLI.

## Get started
```commandline
git clone https://github.com/fleshpriest/Cloud-Crafter
pip install -r requirements.txt
python main
```
Follow the prompts and get a recipe with volumes, weights, & percentages output.
